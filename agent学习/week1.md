人工智能的实现可以通过机器学习（也可以通过规则、搜索、规划等传统方法，而机器学习是目前实现AI的重要方法），深度学习属于当下机器学习中一种高效的方法，而大语言模型就是机器进行深度学习后的产物，同时智能体在接受到外界刺激时，可以借助大语言模型来理解并做出相应的反应。

# 人工智能（AI）

一种完成通常需要人类只能的任务的机器，可以分析事物、做出选择等。

如：Gemini、ChatGPT、元宝、豆包、deepseek

# 机器学习（ML）

通过提供的数据训练，以实现预测、分类、生成、制定决策等功能。

## 分类

### 监督式学习

给出的数据知道其对应的结果，即有特征也有标签。

分为：回归、分类（可以是 yes-or-no，也可以是多种类别）

如：房产预测、雨量预测。

### 非监督式学习

给出的数据没有对应的结果，即有特征无标签，模型需要自己寻找数据中的结构和相似性或分组。

通常只能形成簇，而簇需要人解释。

如：机器将消费者分成若干组，由人分析和命名。

### 强化学习

通过奖惩来调整操作，类似于模仿生物趋利避害的本能。

如：AlphaGo。

# 深度学习（DL）

利用一层层神经网络，让机器用数据训练自己，通过调整网络上的参数，来让预测的输出结果贴近真实的输出结果。

如：利用神经网络识别图片中是否有猫。

# 大语言模型（LLM）

通过自监督训练，以大量文本作为数据来训练的模型，旨在理解和生成自然语言。

可能经过微调和人类的反馈训练。

如：deepseek

# 智能体（Agent）

在理解用户需求后，调用工具自主决策、行动以满足用户需求的机器。

如：codex，Claude Code。
